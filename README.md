# Power BI Doc Fácil — downloads oficiais

## Objetivo deste repositório

Este repositório existe para que **qualquer pessoa** possa **baixar o instalador** do aplicativo **Power BI Doc Fácil** para Windows, de forma **pública**, **gratuita** e **rastreável** (cada versão fica em [Releases](./releases), com notas e histórico).

Aqui **não** há código-fonte nem desenvolvimento do produto — só os **pacotes prontos para instalar** e o que for descrito em cada release. O desenvolvimento do software ocorre em outro lugar; **este endereço é o canal oficial de distribuição** do instalador publicado pelo autor.

---

## O que é o Power BI Doc Fácil

É um **aplicativo para Windows** que ajuda a **gerar documentação técnica** a partir de arquivos **`.pbix`** do Power BI (com suporte a recursos como provedores de IA e configurações locais, conforme a versão). Use-o se você trabalha com modelos Power BI e quer documentar o que há dentro do arquivo de forma estruturada.

---

## Como baixar

1. Abra a página [**Releases**](./releases) (versões publicadas).
2. Entre na **versão mais recente** (no topo da lista).
3. Na seção **Assets**, baixe o arquivo **`.exe`** do instalador indicado na descrição daquela versão (o nome costuma incluir a versão, por exemplo `Power-BI-Doc-Facil-Setup-1.0.0.exe`).

Não é necessário ter conta no GitHub para baixar os arquivos de uma release **pública**.

---

## Requisitos e ambiente

- **Sistema:** Windows compatível com o instalador oferecido na release.
- **Uso do app:** em geral será necessário ter o **Power BI Desktop** instalado e, conforme a documentação da versão, ferramentas auxiliares (por exemplo DAX Studio ou Graphviz) — veja sempre as **notas da release** que você estiver instalando.

---

## Na primeira instalação ou execução

O Windows pode mostrar avisos do **SmartScreen** ou de política de aplicativos para software que **ainda não é amplamente conhecido** ou **não está assinado com certificado comercial**. Isso é comum em projetos independentes. Se você confia na **fonte oficial** (este repositório ou o site do projeto), pode seguir as opções que o próprio Windows oferece (por exemplo **Mais informações** → **Executar assim mesmo**), quando aplicável.

---

## Segurança e fonte oficial

- Baixe o instalador **somente** deste repositório oficial **`powerbi-doc-facil-releases`** ou de links divulgados no **site do projeto** — evite cópias em sites de terceiros que você não conhece.
- Se a release publicar **hash** (por exemplo SHA-256) do arquivo, você pode comparar com o arquivo baixado para confirmar integridade.
